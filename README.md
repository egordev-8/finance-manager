Personal Finance Manager

A simple Python application for tracking personal finances.

Features

* Add income
* Add expenses
* View transaction history
* Calculate current balance
* Store data using SQLite

Technologies

* Python 3
* SQLite

Menu

1. Add income
2. Add expense
3. Show transactions
4. Show balance
5. Exit

Example

Add income:

Income source: Salary

Amount: 1000

Add expense:

Expense category: Food

Amount: 250

Show balance:

Current balance: 750.0

Run

python3 main.py
